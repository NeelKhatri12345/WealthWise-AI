# Analytics Reports

Generated analysis reports, model evaluation summaries, and data quality assessments.

## Contents

Reports are organized by type:

| Subdirectory | Description |
|--------------|-------------|
| Model evaluation reports | Accuracy, precision, recall, F1, confusion matrices |
| Data quality reports | Completeness, consistency, validity checks |
| Pipeline run logs | Automated pipeline execution summaries |

Reports are generated by notebooks and pipeline scripts — do not edit manually.

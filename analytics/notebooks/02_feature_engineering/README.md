# 02 — Feature Engineering

## Purpose
Transforms cleaned financial transaction data into ML-ready features: spending ratios, category aggregations, temporal patterns, and derived financial indicators.

## Inputs
| File | Source |
|------|--------|
| Cleaned transaction data | `analytics/data/cleaned/` |

## Outputs
| File | Destination |
|------|-------------|
| `features_dataset.csv` | `analytics/data/exports/features/` |

## Generated Artifacts
- Engineered feature matrix with 50+ financial indicators
- Feature importance rankings
- Correlation analysis visualizations

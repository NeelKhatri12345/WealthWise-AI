# 06 — AI Financial Coach

## Purpose
Builds the knowledge base and recommendation logic for the AI-powered financial coaching module. Generates personalized advice based on spending patterns, health scores, and risk profiles.

## Inputs
| File | Source |
|------|--------|
| `features_dataset.csv` | `analytics/data/exports/features/` |
| `health_score_dataset.csv` | `analytics/data/exports/health_score/` |
| `risk_profile_predictions.csv` | `analytics/data/exports/risk_profile/` |
| `portfolio_recommendation_dataset.csv` | `analytics/data/exports/portfolio/` |

## Outputs
| File | Destination |
|------|-------------|
| `ai_financial_coach_dataset.csv` | `analytics/data/exports/ai_coach/` |
| `ai_financial_coach_dataset.xlsx` | `analytics/data/exports/ai_coach/` |

## Generated Artifacts
- Coach recommendation dataset with categorized advice
- Spending pattern insights
- Personalized financial action items
